# JavaScript Practice Projects 🚀

This repository contains a collection of JavaScript practice projects and mini-games. Each folder focuses on different concepts, ranging from fundamentals to interactive applications.

---

## 📂 Project Structure

### 1. JavaScript Fundamental Practice Files

This folder includes basic JavaScript exercises and practice files to build a strong foundation.
Topics may include:

* Variables and data types
* Functions
* Loops and conditionals
* Arrays and objects

---

### 2. JS Regex Forms

This section focuses on form validation using JavaScript Regular Expressions.
Key features:

* Input validation
* Pattern matching
* Email, password, and custom validation logic

---

### 3. Guess My Number Game 🎯

A simple interactive number guessing game.
Features:

* Random number generation
* User input handling
* Score tracking
* Feedback messages (too high / too low)

---

### 4. Pig Game 🎲

A fun two-player dice game built with JavaScript.
Features:

* Turn-based gameplay
* Score accumulation
* Risk vs reward mechanics
* Game reset functionality

---

## 🛠️ Technologies Used

* HTML5

* JavaScript (Vanilla JS)
